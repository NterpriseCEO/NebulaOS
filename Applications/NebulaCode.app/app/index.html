On Start

Once you have either created or opened a project, you can do many things.


Click the little plus sign to create new tabs.




If you have already opened a folder you can select and open a file from the file tree.




Once opened, files can be saved using ctrl+s and tabs can be closed. If you are designing an app, you can open it from the file menu by selecting "Open As App" or by using ctrl+r.
